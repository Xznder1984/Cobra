# Cobra Language for VS Code

Provides syntax highlighting, snippets, and language support for the Cobra programming language (`.cb` files).

## Features

- **Syntax Highlighting**: Full TextMate grammar with scoped highlighting for keywords, types, strings, numbers, comments, operators, and function declarations.
- **Snippets**: Ready-to-use snippets for common Cobra patterns including `fn`, `fnmain`, `let`, `var`, `if`/`elif`/`else`, `while`, `for`, `struct`, `class`, `enum`, `trait`, `impl`, `match`, and `println`.
- **Language Configuration**: Bracket matching, auto-closing pairs, comment toggling, and indentation rules.

## Usage

1. Install the extension
2. Open any `.cb` file — Cobra language features activate automatically
3. Start typing Cobra code with full syntax highlighting and snippet support
