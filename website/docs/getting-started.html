Getting Started with Cobra

Welcome to Cobra! This guide will help you install the Cobra compiler, create your first program, and learn the basic workflow.

Installation

Prerequisites

  • macOS or Linux
  • A C compiler (Clang or GCC)
  • Make
  • Git (optional, for building from source)

Option 1: Install via Homebrew (macOS)

brew install Xznder1984/cobra/cobra

Option 2: Install via script

curl -fsSL https://raw.githubusercontent.com/Xznder1984/Cobra/main/install.sh | sh

Option 3: Build from source

git clone https://github.com/Xznder1984/Cobra.git
cd Cobra
make build

Verify installation

cobra --version

Your First Cobra Program

Create a file called hello.cb:

fn main() {
    println("Hello, World!")
    println("Welcome to Cobra!")
}

Run it

cobra hello.cb

Behind the scenes, cobra hello.cb:

  1. Compiles .cb source to assembly (via cobrac)
  2. Assembles with Clang/GCC
  3. Links into a native executable
  4. Runs it

CLI Usage

CommandDescription
cobraStart interactive REPL
cobra file.cbCompile and run a Cobra file
cobra buildBuild project (looks for src/main.cb)
cobra --versionShow version
cobra --helpShow help

Building and Running

Build a project

cobra build

This looks for src/main.cb (or main.cb) and produces build/program.

Run a single file

cobra hello.cb

Compiles and runs hello.cb in one step. The binary is left at build/hello.

REPL (Interactive Mode)

cobra

Starts an interactive REPL for experimentation:

Cobra 0.1.0
Type 'exit()' to quit.

>>> "Hello" + " World"
Hello World
>>> 2 + 2
4
>>> exit()

Troubleshooting

cobrac: command not found

Ensure the compiler binary is in your PATH:

export PATH="$PATH:/usr/local/bin"

Build fails with assembler errors

Make sure Clang or GCC is installed:

# macOS
xcode-select --install

# Ubuntu/Debian
sudo apt-get install build-essential

# Fedora
sudo dnf install gcc
