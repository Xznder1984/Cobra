Why Cobra?

Designed for modern software development — combining Python's elegance with Rust's performance and safety guarantees.

⚡

Blazing Speed

Compiles to native machine code via LLVM, delivering performance on par with C and Rust. No runtime overhead, no garbage collector pauses.

✨

Pythonic Syntax

Familiar, readable syntax inspired by Python. Indentation-based blocks, dynamic feel with static types — the best of both worlds.

🛡️

Memory Safety

An ownership-based type system eliminates entire classes of bugs — no null pointer dereferences, no use-after-free, no data races.

⚙️

Zero-Cost Abstractions

High-level constructs compile down to optimal machine code. You pay only for what you use — no hidden overhead from abstractions.

🔗

Seamless Interop

Call C libraries directly with zero boilerplate. First-class FFI support means you can use any C, C++, or Rust library in your Cobra projects.

🚀

Built-in Concurrency

Async/await out of the box with a lightweight task system. Write concurrent code that's as easy as Python's asyncio — but 100x faster.

# Hello, World! in Cobra
fn main():
    print("Hello, World!")

# Functions are defined with 'fn'
# Indentation-based blocks (like Python)
# Strong static typing with inference
# Fibonacci with pattern matching
fn fib(n: int) -> int:
    match n:
        0 => return 0
        1 => return 1
        _   => return fib(n - 1) + fib(n - 2)

fn main():
    for i in 0..10:
        print("fib({}) = {}".format(i, fib(i)))
// Struct with methods in Cobra
struct Vec3:
    x: f64
    y: f64
    z: f64

    // Method on Vec3
    fn magnitude(self) -> f64:
        return (self.x**2 + self.y**2 + self.z**2).sqrt()

    fn normalize(self) -> Vec3:
        mag := self.magnitude()
        return Vec3(self.x/mag, self.y/mag, self.z/mag)

fn main():
    v := Vec3(3.0, 4.0, 0.0)
    print(v.magnitude())  // 5.0
# Async HTTP server in Cobra
use net.http

async fn handle_request(req: Request) -> Response:
    match req.method, req.path:
        "GET", "/" =>
            return Response(200, "Hello, Cobra!")
        "GET", "/api" =>
            data := await fetch_database()
            return Response.json(200, data)
        _ =>
            return Response(404, "Not found")

fn main():
    server := http.Server(port: 8080)
    server.handle(handle_request)
    server.listen()
